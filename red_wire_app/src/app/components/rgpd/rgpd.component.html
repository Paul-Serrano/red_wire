Red Wire est une application web qui vous fournit des informations sur votre localisation et la météo environnante. Vous pouvez également consulter votre historique.

Développée selon une logique de mobile first, elle utilise un backend Python, Angular JS pour la partie frontend, et Tailwind CSS pour le design. Les données sont stockées dans une base de données MongoDB.

Elle se connecte à deux API : Ipinfos pour les données de géolocalisation et OpenWeather pour les données météorologiques.
